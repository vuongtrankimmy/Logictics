# Địa Điểm API v3.0


### Cập nhật


## ``FRONTEND USER *[17/06/2020]*``

• **Discounts**: Cập nhật thông tin khuyến mãi
      
*  **ListDiscounts**: Danh sách khuyến mãi hiển thị dựa vào khoảng thời gian DateOn và DateOff

                 **DateRange**: 1: Áp dụng thời gian bắt đầu(DateOn) và kết thúc(DateOff) khuyến mãi. **0**: nhập mã khuyến mãi để áp dụng.
                 **DateOn**: Thời gian bắt đầu áp dụng khuyến mãi.
                 **DateOff**: Thời gian kết thúc áp dụng khuyến mãi.
                 **SetSchedules**: **1**: Áp dụng tắt mở thời gian áp dụng khuyến mãi theo ngày. **0**: nhập mã khuyến mãi để áp dụng.
                 
*   **ListDiscountsCondition** : Điều kiện khuyến mãi gồm: %, Số tiền, Miễn phí vận chuyển, Buy X get Y, Combo,....
                  
                DiscountGuid : Mã khuyến mãi
                DiscountType: Loại khuyến mãi

*   **ListDiscountsImages** : Hình ảnh khuyến mãi hiển thị.
*   **ListSystemDiscountType** : Loại khuyến mãi.
